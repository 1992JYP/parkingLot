1. Take_Parking_pic.py 실행
2. 카메라에서 촬영한 'Parking_Zone_Pic.jpg' 파일 생성

3. Select_Parking_Area.py 실행
4. 'Parking_Zone_Pic.jpg' 파일을 불러와
   해당 파일 사진에서 주차칸 영역을 지정
5. 지정이 끝나면 'CarParkPos' 이름의 파일 생성하여
   지정한 주차칸 정보를 저장

6. Main.py 실행
7. 카메라 화면을 화면에 출력
8. 'CarParkPos' 파일을 불러와 지정한 주차칸의 영역을
   카메라 화면에 덧씌워서 출력
9. 지정한 영역의 색 변화를 탐지하여
   주차 가능 여부 판단
